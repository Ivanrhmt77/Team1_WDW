Api Abadi

Mojokerto,Jawa Timur,Indonesia

Api abadi Mojokerto adalah sebuah fenomena alam yang terjadi akibat keluarnya gas alam dari celah batuan atau tanah yang terbakar sehingga tercipta api yang tak pernah padam. Api ini berlokasi di Dusun Bekucuk, Desa Tempuran, Kecamatan Soko, Mojokerto, Jawa Timur. Api ini diperkirakan sudah ada sejak zaman Kerajaan Majapahit dan pernah digunakan untuk menempa besi oleh para ahli dalam pembuatan senjata atau benda pusaka. Untuk mencapai api abadi Mojokerto dari pusat kota Mojokerto, Anda bisa menggunakan kendaraan pribadi atau umum. Jarak tempuh menuju lokasi sekitar 3 km dari arah Kota Mojokerto. Anda bisa mengikuti peta yang tersedia di Google Maps untuk menemukan lokasi api abadi ini. Api abadi Mojokerto berada di sebelah kanan jalan raya, di dekat sebuah kolam. Anda bisa melihat api abadi ini dengan jelas pada malam hari, karena api akan bersinar lebih terang di kegelapan.
